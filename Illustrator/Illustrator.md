# Illustrator
### 일러스트레이터란?
> **벡터값으로 이뤄진 파일**을 다루는 디자인 도구

### 선 도구, 획 패널

### 색상
1. RGB
	- 디지털 그래픽의 색상 모델
	- 웹 작업에 적합
	- 0(검은색) ~ 255(흰색)까지의 값을 가짐
2. CMYK
	- 인쇄물에 적합한 색상 모델
3. HSB

### 마우스 이벤트
대상 복제: alt + 드래그
